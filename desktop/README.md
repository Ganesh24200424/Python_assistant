# Jarvis Desktop

Desktop application for J.A.R.V.I.S AI Assistant built with Electron + React.

## Setup

```bash
cd desktop
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build:win
```

## Requirements

- Backend server must be running: `python run.py` from the root directory
- Open http://localhost:8000 in browser to verify backend is working
